# Design Library

This folder contains design specifications and style guides for reference when creating designs.

## Files

- **design-spec-hr-document-tracking.uson** - USON design specification for HR Document Expiration Tracking system
  - Format: A4 landscape
  - Style: Modern landing page inspired by Markova SaaS platform
  - Color scheme: Primary blue (#4461F2), success green, danger red
  - Layout: Three-column grid with problem/solution/results sections

## Usage

When creating designs for this project, reference these specifications to maintain consistency in:
- Color palettes
- Typography
- Layout patterns
- Component styles
- Spacing and margins

## USON Format

USON (Unified Style Object Notation) is a structured format for defining design specifications. When implementing designs, translate these specifications into actual CSS/HTML/design code while maintaining the visual style and layout described.
